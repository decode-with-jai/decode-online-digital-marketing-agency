## 💡 Value Proposition

### ❌ Problem
Most agency websites look outdated, cluttered, and fail to guide users toward conversion.

### ✅ Solution
Decode Online delivers a **modern, minimal, and conversion-driven UI** with:
- Sticky sidebar navigation  
- Smooth scrolling experience  
- Clean layout for better readability  

### 🎯 Outcome
- Higher user engagement  
- Better navigation flow  
- Professional digital presence  
- Improved conversion potential  

---

## 🧠 About the Project

Decode Online is a **frontend-based digital marketing agency website** built to showcase services in a modern, structured, and visually appealing way.

✨ Built under **Greenrootz Media**
